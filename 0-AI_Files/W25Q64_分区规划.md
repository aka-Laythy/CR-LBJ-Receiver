# W25Q64 存储分区规划

容量：64 Mbit = **8 MBytes**（地址 0x000000 ~ 0x7FFFFF）

| 分区 | 起始地址 | 大小 | 说明 |
|------|---------|------|------|
| 系统配置 | `0x000000` | **16 KB** | 固件参数、校准值、偏好设置等 |
| 预留 | `0x004000` | **1 MB** | 后续功能扩展（如波形记录、日志等） |
| 中文字库 | `0x104000` | **256 KB** | GB2312 16×16 点阵字库（~6763 字 × 32 字节 ≈ 211 KB + ASCII） |
| LBJ 数据记录 | `0x144000` | **~6.6 MB** | 列车接近预警消息存储 | 

## 各分区详情

### 1. 系统配置（16 KB）
- 扇区：0~3（4 个扇区 × 4 KB）
- 擦除粒度：每次写入前擦除整个分区
- 内容：运行参数、用户设置等

### 2. 预留（1 MB）
- 扇区：4~255（252 个扇区）
- 未分配用途

### 3. 中文字库（256 KB）
- 扇区：256~319（64 个扇区）
- 字库组织：
  - ASCII 字符：0x20~0x7E，每字 16 字节（8×16 点阵）
  - 中文 GB2312：区码 16~55（一级汉字），每字 32 字节（16×16 点阵）
  - 编码方式：`((区码 - 0xA1) * 94 + (位码 - 0xA1)) * 32`
- 数据来源：从 OLED 现有字库提取或转换标准 UCDOS HZK16

### 4. LBJ 数据记录（~6.6 MB）
- 扇区：320~2047（1728 个扇区）
- 记录格式（待定义）：
  ```
  时间戳(4B) + RIC(3B) + 车次号(6B) + 速度(3B) + 公里标(5B) + 机车代码(3B) + 机车号(5B) + 线路名(16B) + 方向(1B) + 校验(1B)
  = 约 48 字节/条
  ```
- 容量估计：6.6 MB / 48 字节 ≈ **140,000+ 条记录**
